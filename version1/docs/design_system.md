# Design System

## Structural Components

- Header: Contains the game title.
- Footer: Contains the copyright information.
- Game Board: Displays the 3x3 grid for the game.

## Color Palette

- Background: #f0f0f0
- Primary: #333
- Secondary: #ccc
- Accent: #ff9800

## Fonts and Sizes

- Body Font: Arial, sans-serif
- Header Font Size: 24px
- Cell Font Size: 2em
